# TT Ауто Контрол - Веб сајт

Ова е веб сајт за TT Ауто Контрол, компанија за технички преглед и регистрација на возила.

## Структура на проектот

- `index.html` - Главна HTML страница
- `style.css` - CSS стилови
- `script.js` - JavaScript функционалност
- `images/` - Директориум за слики
- `README.md` - Документација на проектот

## Технологии

- HTML5
- CSS3
- JavaScript
- Google Maps API
- Font Awesome икони

## Инсталација

1. Клонирајте го репозиториумот
2. Отворете го `index.html` во веб прелистувач
3. За да работи Google Maps функционалноста, додадете го вашиот API клуч во `script.js`

## Развој

За да работите на проектот:

1. Инсталирајте го Git
2. Клонирајте го репозиториумот
3. Направете ги потребните промени
4. Комитнете ги промените
5. Пуштете ги промените на серверот
